# SPB Guide — инструкции для Cursor

Это сайт-гид по местам в Санкт-Петербурге.

Пользователь может выбрать по фильтрам, что он хочет найти:
- поесть
- выпить
- погулять
- развлечься
- посмотреть красивое место
- найти атмосферную локацию

После выбора пользователю показываются карточки подходящих мест.

## Первая версия сайта

Пока делаем только публичную часть сайта:

- главная страница
- фильтры по типу места
- карточки мест
- тестовые данные внутри проекта
- адаптивная версия для всех устройств

Пока НЕ делаем:

- базу данных
- Supabase
- авторизацию
- админку
- загрузку фото
- деплой
- сложную архитектуру

## Будущая версия

Позже нужно будет добавить:

- админку по адресу /admin
- логин только для владельца
- добавление новых мест через форму
- редактирование мест
- удаление мест
- базу данных Supabase
- хранение фото

## Технологии

Проект уже создан на:

- Next.js
- TypeScript
- Tailwind CSS
- App Router

Структура проекта сейчас без папки src, поэтому основные папки должны быть в корне проекта:

- app/ — страницы сайта
- components/ — переиспользуемые компоненты
- data/ — временные тестовые данные
- types/ — TypeScript-типы
- lib/ — вспомогательные функции, если понадобятся
- public/ — статичные файлы и изображения

## Важные правила

- Не усложнять архитектуру без необходимости.
- Делать маленькими понятными шагами.
- Не подключать новые библиотеки без объяснения.
- Не подключать базу данных, авторизацию и админку без отдельной задачи.
- Сначала всегда объяснять план, потом менять код.
- После изменений кратко объяснять, какие файлы были изменены.
- Писать код так, чтобы начинающий человек мог примерно понимать структуру проекта.
